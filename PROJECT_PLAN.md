## 🏗 Архитектура

### Страницы

| Страница | Роут | Назначение |
|---|---|---|
| Главный экран | `/dashboard` | Workflow, таймер, статистика, лог |
| Проекты | `/projects` | Редактирование проектов / подпроектов / задач |
| Умное добавление | `/smart-add` | Мультиввод + автоструктурирование |
| Профиль | `/profile` | Настройки, статистика, друзья |

### База данных

| Сущность | Описание |
|---|---|
| **Users** | Пользователи системы |
| **Groups** | Группы пользователей |
| **Contacts** | Контакты / друзья |
| **Tags** | Теги для задач |
| **Quests** | Квесты |
| **Quest Types** | Типы квестов (daily / total_time / task_completion) |
| **Projects** | Проекты (уровень 1 иерархии) |
| **Subprojects** | Подпроекты (уровень 2 иерархии) |
| **Tasks** | Задачи (уровень 3 иерархии) |

### Ключевые фичи (сводка)

- [ ] **Умный таймер** — бесшовный переход между задачами, тянет статистику
- [ ] **Workflow** — настраиваемая последовательность задач с учетом времени и приоритетов
- [ ] **Иерархия задач** — 3 уровня: project → subproject → task + теги
- [ ] **Авторазбиение** — AI / rule-based разбивка формулировок, тегирование
- [ ] **Мультиввод** — текст, голос, изображения
- [ ] **Квесты** — 3 типа, награды за выполнение
- [ ] **Рейтинг задач** — приоритет × сложность × просрочка
- [ ] **Лог прогресса** — свой + друзей, дневной скор
