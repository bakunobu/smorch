## 🏗 Архитектура

### Страницы

| Страница | Роут | Назначение |
|---|---|---|
| Главный экран | `/dashboard` | Workflow, таймер, статистика, лог |
| Проекты | `/projects` | Редактирование проектов / подпроектов / задач |
| Умное добавление | `/smart-add` | Мультиввод + автоструктурирование |
| Профиль | `/profile` | Настройки, статистика, друзья |

### API Endpoints

| Endpoint | Метод | Назначение | Статус |
|---|---|---|---|
| `/api/health` | GET | Проверка работоспособности | ✅ Готов |

### База данных

#### Реализованные сущности

| Сущность | Таблица | Описание |
|---|---|---|
| **Users** | `users` | Пользователи системы (имя, никнейм, email, хеш пароля, логическое удаление) |
| **Projects** | `projects` | Проекты (уровень 1 иерархии) |
| **Subprojects** | `subprojects` | Подпроекты (уровень 2 иерархии, FK → projects) |
| **Tasks** | `tasks` | Задачи (уровень 3 иерархии, FK → subprojects; priority, complexity, deadline, completed) |
| **Tags** | `tags` | Теги для задач (unique name) |
| **Tag-Task** | `tag_task_association` | Many-to-many связь тегов и задач |
| **Follow** | `follows` | Подписки пользователей друг на друга (follower → followed, unique constraint, индексы) |

#### Планируемые сущности (к реализации)

| Сущность | Описание |
|---|---|
| **Groups** | Группы пользователей |
| **Contacts** | Контакты / друзья (возможно, поверх Follow) |
| **Quests** | Квесты |
| **Quest Types** | Типы квестов (daily / total_time / task_completion) |

### Примечания по моделям

- [`src/models/base.py`](src/models/base.py:1) содержит все модели: `Project`, `Subproject`, `Task`, `Tag`, `User`, `Follow` + ассоциативная таблица `tag_task_association`
- `User` имеет поле `is_deleted` для логического удаления
- `Project`, `Subproject`, `Task` имеют поле `is_private` (Boolean, default=True) — приватность по умолчанию
- `Follow` использует составной unique constraint (`follower_id`, `followed_id`) для предотвращения дублирующихся подписок
- [`src/models/__init__.py`](src/models/__init__.py:1) экспортирует только `Project, Subproject, Tag, Task` — **не экспортирует `User` и `Follow`** (возможная проблема)
- [`src/__init__.py`](src/__init__.py:1) экспортирует `Project, Subproject, Tag, Task, User` — но не `Follow`
